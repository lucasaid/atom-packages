# Atom Packages

Just a list of the atom packages I have installed, could probably remove some of them to be less cluttered.  

#### [KeyCount](https://atom.io/packages/KeyCount)  
Simple tool to count the number of key strokes using Atom.  

#### [Zen](https://atom.io/packages/Zen)  
Distraction free writing for Atom.

#### [activate-power-mode](https://atom.io/packages/activate-power-mode)  
Activate POWER MODE to write your code in style.

#### [angularjs](https://atom.io/packages/angularjs)  
An AngularJS package for Atom.

#### [atom-autocomplete-php](https://atom.io/packages/atom-autocomplete-php)  
Atom autocompletion plugin for PHP language.

#### [atom-beautify](https://atom.io/packages/atom-beautify)  
Beautify HTML, CSS, JavaScript, PHP, Python, Ruby, Java, C, C++, C#, Objective-C, CoffeeScript, TypeScript, Coldfusion, and SQL in Atom.

#### [atom-bootstrap3](https://atom.io/packages/atom-bootstrap3)  
Bootstrap 3 Support with snippets and autocompletion. Supports Glyphicons and Fontawesome.

#### [atom-html-preview](https://atom.io/packages/atom-html-preview)  
A live preview tool for Atom Editor.

#### [atom-jade](https://atom.io/packages/atom-jade)  
Jade language support in Atom Editor.

#### [atom-lorempixel](https://atom.io/packages/atom-lorempixel)  
Insert an img tag with a placeholder image from lorempixel.com into your code.

#### [atom-minify](https://atom.io/packages/atom-minify)  
Minifies JS and CSS files, optionally on save.

#### [atom-spotify2](https://atom.io/packages/atom-spotify2)  
Displays the currently playing song in Spotify in Atom's status bar.

#### [auto-detect-indentation](https://atom.io/packages/auto-detect-indentation)  
Automatically detect indentation of opened files.

#### [autoclose-html](https://atom.io/packages/autoclose-html)  
Automates closing of HTML Tags.

#### [autoclose-html](https://atom.io/packages/autoclose-html)  
Automates closing of HTML Tags.

#### [autocomplete-paths](https://atom.io/packages/autocomplete-paths)  
Adds path autocompletion to autocomplete+.

#### [autocomplete-plus](https://atom.io/packages/autocomplete-plus)  
Display possible completions in the editor while typing.

#### [autoprefixer](https://atom.io/packages/autoprefixer)  
Prefix CSS and SCSS.

#### [blade-snippets](https://atom.io/packages/blade-snippets)  
Laravel 5 blade snippets! useful snippets for the blade templating engine.

#### [caniuse](https://atom.io/packages/caniuse)  
Quick access to the caniuse.com support tables.

#### [color-picker](https://atom.io/packages/color-picker)  
A Color Picker for Atom.

#### [csscomb](https://atom.io/packages/csscomb)  
Atom Editor plugin for CSScomb, http://csscomb.com.

#### [csslint](https://atom.io/packages/csslint)  
CSSLint error reports for your Atom editor.

#### [dash](https://atom.io/packages/dash)  
Dash documentation integration with Atom.

#### [docblockr](https://atom.io/packages/docblockr)  
A helper package for writing documentation.

#### [emmet](https://atom.io/packages/emmet)  
Emmet – the essential tool for web developers.

#### [es6-javascript](https://atom.io/packages/es6-javascript)  
A collection of commands and ES6 focused snippets for optimizing modern Javascript development productivity.

#### [file-icons](https://atom.io/packages/file-icons)  
Assign file extension icons and colours for improved visual grepping.

#### [git-time-machine](https://atom.io/packages/git-time-machine)  
isually interact with git commit history for a file.

#### [gulp-control](https://atom.io/packages/gulp-control)  
Displays a list of gulp tasks and allows execution within Atom.

#### [highlight-line](https://atom.io/packages/highlight-line)

#### [highlight-selected](https://atom.io/packages/highlight-selected)

#### [javascript-snippets](https://atom.io/packages/javascript-snippets)  
JavaScript & NodeJS Snippets for Atom.

#### [language-blade](https://atom.io/packages/language-blade)
